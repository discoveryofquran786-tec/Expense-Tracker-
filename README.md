# THE H - Professional
**By Abdul Rahman**

A professional voice-controlled finance tracker with Hinglish support.

### Features
* **Logo Integrated:** "AR Abdul Rahman Professional" branding.
* **Hinglish Voice:** Uses 'en-IN' to write Hindi words in English text.
* **Smart Deletion:** Trash icon to remove wrong entries.
* **Auto-Archive:** Checks date on load to save yesterday's file.
* **Manual Export:** Download your data anytime as CSV.
